# comandos-unixx
Los comandos mas utilizados en unix son: 
Is: sirve para listar los nombres de los archivos en le directorio actual (terminal)
cat: se utiliza para imprimir contenido de uno o varios archivos en la terminal
cp: permite copiar archivos de un destino a otro
mv: se emplea para mover archivos o renombrarlos 
rm: elimina archivos de forma permanenete 
pwd: nos muestra lo que hemos hecho actualmente en la terminal o directorio de trabajo (print working directory)
cd: comando indispensable para cambiar de un directorio a otro
mkdir y rmdir: se usan para crear o eliminar directorios

grep: busca patrones especificos de texto dentro de uno o mas archivos 
wc: cuenta el numero de lineas, palabras y caracteres en un archivo
sort: ordena las lineas de un archivo alfabetica y numericamente
tail: imprime las ultimas lineas en un archivo
diff: compara los archivos y senala todas sus diferencias linea por linea 
sed y awq: son herramientas mas avanzadas para la edicion y procesamiento de flujo de datos. 
who: muetsra quien esta utilizando la maquina en ese momento
date: informa la fecha y la hora actual
man: es el acceso al manual en linea, permite entender el funcionamiento de cualquier herramienta
ps: proporciona informacion sobre los procesos que se estan ejecutando actualmente
kill: se utiliza para detener o finalizar procesos mediante su numero de identificacion
mail: permite enviar y recibir mensajes entre usuarios de sistema 
ed: es el editor de texto estandar de UNIX
